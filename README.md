# Private Health & Fitness Coach with On-Device AI

## Project Overview
Private Health & Fitness Coach is a privacy-focused health assistant web application powered by **on-device AI using the RunAnywhere SDK**. The application helps users manage their physical and mental wellness through meal analysis, mood tracking, personalized workout generation, and voice interaction.

All AI processing runs directly on the user's device instead of cloud servers. This ensures **privacy, low latency, zero cloud cost, and offline capability**, making the application fast, secure, and accessible anywhere.

---

## Key Features

### User Profile & Onboarding
Users first complete a profile setup by entering:
- Name  
- Age  
- Height  
- Weight  
- Fitness goals  

This information is used to generate personalized health recommendations.

---

### Meal Analysis
Users can upload a **photo of their meal**, and the AI analyzes it to provide:
- Estimated calories  
- Protein content  
- Carbohydrates  
- Nutritional insights  

This helps users understand the **nutritional composition of their food**.

---

### Mood Tracking & Mental Wellness
The app allows users to track:
- Daily mood
- Energy level
- Activities
- Personal notes

The AI analyzes mood patterns and provides **wellness suggestions and relaxation activities**.

---

### Personalized Workout Generator
Based on the user's profile and fitness goals, the application generates **custom workout routines** to support a healthier lifestyle.

---

### Voice Assistant
The application supports voice interaction using:
- **Speech-to-Text (STT)** for voice input
- **Text-to-Speech (TTS)** for spoken responses

Users can interact with the app hands-free through voice commands.

---

## On-Device AI Benefits

### Privacy
Sensitive user data such as meal images, mood logs, and personal health details remain **on the user's device** and are not sent to external servers.

### Low Latency
AI responses are generated **quickly** because processing happens locally on the device.

### Zero Cloud Cost
The application does not rely on external AI APIs or cloud services, eliminating **server and API costs**.

### Offline Capability
Once the AI models are loaded, the application can **work even without an internet connection**.
